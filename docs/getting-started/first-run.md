# 최초 실행

`PongDu.exe`를 실행하면 나오는 게이트 화면(런처)의 흐름을 순서대로 설명합니다.

!!! note "중복 실행 방지"

    이미 퐁듀가 실행 중인 상태에서 한 번 더 실행하면 "이미 실행 중입니다" 경고가 뜨고 종료됩니다. 실행파일 여러 개를 동시에 띄울 수 없습니다.

## 1단계 — 치지직 채널 확인

첫 화면에서 본인의 치지직 채널을 입력합니다. 아래 세 가지 형식 모두 지원합니다.

- 채널 URL (`https://chzzk.naver.com/live/…`)
- 채널명
- 채널 UUID

입력 후 **확인** 버튼을 누르면 서버가 채널을 조회하고, 이번 시즌 화이트리스트에 등재되어 있는지 확인합니다.

!!! failure "유효하지 않은 채널입니다"

    아래 경우에 이 오류가 뜹니다.

    - 입력한 값으로 채널을 특정할 수 없음
    - 채널은 찾았지만 화이트리스트에 없음 (시즌 미참가 채널)
    - 화이트리스트 원격 조회 자체가 실패함 (안전을 위해 이 경우도 차단 처리됩니다)

    **다시입력** 버튼으로 처음 화면으로 돌아가 재시도할 수 있습니다.

## 2단계 — 연동 체크리스트

채널 확인에 성공하면 아래 4가지 항목을 3초 간격으로 자동 갱신하며 보여줍니다. 모두 초록색이 되어야 다음으로 넘어갈 수 있습니다.

| 항목             | 의미                                 |
| ---------------- | ------------------------------------ |
| 채널 확인        | 1단계에서 이미 통과됨                |
| 생방송 중        | 현재 방송이 켜져 있는지              |
| PZ 실행 중       | Project Zomboid 프로세스가 떠 있는지 |
| 인게임 접속 완료 | PZ 서버에 실제로 접속해 있는지       |

!!! warning "연령제한 감지"

    이 단계에서 방송이 19세로 전환된 것이 감지되면 팝업 경고가 뜨고 **연동 시작** 버튼이 비활성화됩니다. 일반 방송으로 되돌리면 자동으로 다시 활성화되니 별도 조작은 필요 없습니다.

이 화면에서 리워드 프리셋을 미리 불러올 수도 있습니다.

- **리워드 프리셋 불러오기**: `{금액: featureId}` 형식의 JSON 파일을 선택해 후원 금액별 기능 매핑을 미리 지정
- **초기화**: 저장된 프리셋을 지우고 기본 티어로 복원

모든 체크리스트가 통과되면 **연동 시작** 버튼이 활성화되고, 클릭 시 실제 연동 화면(메인 창)으로 넘어갑니다.

## PZ 원클릭 최적화 (자동)

런처 창이 뜨고 약 0.4초 뒤, 별도의 확인창 없이 PZ 최적화가 자동으로 시도됩니다.

- **최초 실행**: 강제로 최적화를 적용하고 이후에도 자동 유지되도록 설정을 저장합니다.
- **이후 실행**: 게임 업데이트 등으로 패치 상태가 어긋나 있으면 조용히 다시 적용합니다.
- **사용자가 직접 "해제"를 누른 경우**: 그 상태를 기억하고 자동으로 다시 적용하지 않습니다.

!!! tip "PZ 실행 중이면 적용 안 됨"

    게임이 이미 켜져 있으면 최적화를 적용할 수 없습니다. 화면 하단에 "게임 최적화 대기" 문구가 뜨면 게임을 끈 뒤 퐁듀를 재시작하세요.

화면 하단의 상태 문구로 현재 최적화 여부를 확인할 수 있고, 옆의 **적용 / 해제** 버튼으로 언제든 전환할 수 있습니다.
