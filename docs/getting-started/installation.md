# 설치

퐁듀는 별도 설치 프로그램 없이 실행파일 하나(`PongDu.exe`)로 동작하는 포터블 앱입니다.

!!! info "배포 방식 미정"

    아직 공식 배포 채널(GitHub Releases 등)이 정해지지 않았습니다. 추후 업데이트됩니다.

## 파일 배치

`PongDu.exe`는 어느 폴더에 두어도 상관없습니다. 설정과 리워드 프리셋은 실행파일 위치가 아니라 **Project Zomboid 게임 세이브 폴더**(`~/Zomboid` 등)에 저장되므로, exe를 옮기거나 새 버전으로 교체해도 기존 설정이 유지됩니다. 자세한 내용은 [최초 실행](first-run.md)과 [업데이트](update.md) 문서를 참고하세요.
