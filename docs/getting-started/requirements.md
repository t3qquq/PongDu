# 시스템 요구사항

퐁듀(PongDu)를 사용하기 전에 아래 조건을 먼저 확인하세요.

## 운영체제

| 항목 | 요구사항                |
| ---- | ----------------------- |
| OS   | Windows 10 / 11 (64bit) |

!!! warning "Windows 전용"

    PZ 원클릭 최적화 기능(자동 힙 설정, 관리자 권한 재실행)이 Windows API에 의존하고 있어 macOS·Linux에서는 정상 동작하지 않습니다.

## 게임

| 항목      | 요구사항                                              |
| --------- | ----------------------------------------------------- |
| 게임      | Project Zomboid - **기본 공개 버전 (Build 41.78.xx)** |
| 설치 경로 | Steam 버전 권장                                       |

!!! note "B41 전용"

    **퐁듀**는 B41 Lua API를 기준으로 제작되었습니다. B42로 전환한 서버에서는 동작을 보장하지 않습니다.

Steam 버전을 사용하면 퐁듀의 **PZ 원클릭 최적화** 기능이 Steam 라이브러리 폴더(`libraryfolders.vdf`)를 자동으로 스캔해 설치 경로를 찾아줍니다. 비-Steam 버전은 이 자동 탐지가 동작하지 않을 수 있습니다.

## 방송 환경

| 항목      | 요구사항                               |
| --------- | -------------------------------------- |
| 플랫폼    | CHZZK (치지직)                         |
| 방송 상태 | 연동 시작 시 반드시 생방송 중이어야 함 |
| 참가 자격 | 화이트리스트 등재 채널만 연동 가능     |
| 연령 제한 | 19세(성인) 인증 방송은 연동 불가       |

!!! info "화이트리스트란?"

    시즌에 참가하는 스트리머 채널 목록입니다.<br>퐁듀 실행시 해당 목록을 자동으로 받아오며, 화이트리스트에 없는 채널은 연동이 불가능합니다.

19세 방송은 연동 체크리스트 단계에서 자동으로 감지되어 "연동 시작" 버튼이 비활성화됩니다. 일반 방송으로 전환하면 별도 조작 없이 바로 다시 활성화됩니다.

## 기타

- 인터넷 연결 (치지직 API 및 화이트리스트 조회에 사용)
- PZ 원클릭 최적화가 게임 설치 폴더에 접근 권한이 없는 경우(예: `Program Files`), 최초 적용 시 Windows UAC 관리자 권한 승격 창이 뜰 수 있습니다.
