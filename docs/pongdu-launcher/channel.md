# 채널 연동

퐁듀는 별도의 네이버/치지직 로그인 없이, **채널 식별 + 화이트리스트 검증**만으로 연동 대상을 결정합니다.

!!! info "로그인 기능은 없습니다"

    현재 버전은 계정 로그인 없이 공개 정보만으로 채널을 특정합니다. 네이버 계정 로그인(19+ 인증 포함)은 아직 구현되지 않은 예정 기능입니다.

## 채널 입력 형식

런처 화면 또는 메인 창의 채널 입력칸에 아래 세 가지 중 아무거나 넣을 수 있습니다.

- 채널 URL: `https://chzzk.naver.com/live/…`
- 채널명
- 채널 UUID (32자리 hex)

## 화이트리스트 검증

입력한 채널은 매번 원격 화이트리스트와 대조됩니다.

- **캐시 없음** — 확인 버튼을 누를 때마다 최신 목록을 새로 받아옵니다.
- **Fail-closed** — 화이트리스트를 아예 못 받아온 경우에도 안전하게 전원 차단 처리됩니다. 네트워크 문제로 목록 조회가 실패했다고 해서 검증을 건너뛰지 않습니다.
- 화이트리스트에 없는 채널은 UUID가 정상적으로 조회되어도 "유효하지 않은 채널입니다" 오류로 처리됩니다.

## 연동 상태 확인

채널이 인식되면 아래 상태를 실시간(3초 간격)으로 추적합니다.

| 상태 | 설명 |
| --- | --- |
| 생방송 중 | 채널이 현재 방송 중인지 |
| 19세(성인) 여부 | 방송에 연령 제한이 걸려 있는지 |
| PZ 실행 중 | Project Zomboid 프로세스 감지 |
| 인게임 접속 완료 | 실제 서버 접속 여부 |

## 19세(성인) 방송 처리

방송이 19세로 전환되면:

1. 팝업으로 즉시 알림
2. 런처 화면이면 **연동 시작** 버튼 비활성화
3. 메인 창에서 연동 중이었다면 연동을 자동 중단하고 런처 화면으로 복귀

일반 방송으로 되돌아오면 별도 조작 없이 자동으로 다시 정상 처리됩니다.

## 재접속 / 자동 시작

- 채널을 한 번 인식시키고 나면 `chzzk_donation_config.json`에 채널 정보가 저장되어 다음 실행 때 자동으로 채워집니다.
- 런처에서 체크리스트를 모두 통과해 메인 창으로 넘어가면, 메인 창은 뜨자마자(0.3초 뒤) 자동으로 연동을 시작합니다 — 별도로 "연동 시작" 버튼을 다시 누를 필요가 없습니다.
- Project Zomboid 종료가 감지되면 연동 중이던 메인 창은 자동으로 정리되고 런처 화면으로 돌아갑니다.
