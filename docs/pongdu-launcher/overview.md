# 개요

**퐁듀(PongDu)**는 치지직(CHZZK) 실시간 후원을 감지해서 Project Zomboid B41 서버에 전달하는 데스크톱 릴레이 앱입니다. 게임 모드가 아니라 게임 밖에서 돌아가는 별도 프로그램이며, 모드([t3chzzkDonation](../mod-features/overview.md))와 짝을 이뤄 동작합니다.

```
도네이션 → 퐁듀 런처 → rewards.txt → 인게임 모드 → 이벤트 발동
```

## 창 구성

퐁듀는 실행 흐름에 따라 두 개의 창을 순서대로 보여줍니다.

1. **런처(게이트)** — 채널 확인부터 화이트리스트 검증, 생방송·PZ 실행·인게임 접속 체크리스트까지 통과 화면. 자세한 순서는 [최초 실행](../getting-started/first-run.md) 참고.
2. **메인 창** — 실제 연동이 진행되는 화면. 채널/경로 상태, 리워드 티어 편집, 테스트 후원, 실시간 로그가 여기 모여 있습니다.

메인 창에서 연동을 중지하거나, PZ 종료·19세 전환이 감지되면 자동으로 다시 런처(게이트)로 돌아갑니다.

## 설계 원칙 (참고)

퐁듀는 치지직 API 호출부(`ChzzkpySource`)와 게임 출력부(`ZomboidAdapter`)가 분리된 구조입니다. 그 사이를 `DonationWorker`로 연결하여, 다른 게임으로 확장할 여지를 열어두고 디자인하였습니다.
