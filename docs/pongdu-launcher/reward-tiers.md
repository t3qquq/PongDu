# 리워드 티어

후원 금액과 인게임 기능(featureId)을 연결하는 설정입니다. **금액이 정확히 일치해야** 해당 기능이 발동합니다 (이상/이하 구간이 아닙니다).

앱에 내장된 기본값은 아래와 같습니다. 설정 파일이 없는 최초 실행이나 구버전에서 마이그레이션할 때 이 값이 쓰입니다.

| 금액 | 기능 |
| --- | --- |
| 1,000원 | 버프 룰렛 |
| 2,000원 | 디버프 룰렛 |
| 3,000원 | 좀비 룰렛 |
| 7,000원 | 백신 |
| 10,000원 | 스프린터 5마리 |
| 20,000원 | 적대 NPC (근접) |
| 30,000원 | 신체 강화 혈청 |
| 40,000원 | 특수좀비 소환 |
| 50,000원 | 적대 NPC (원거리) |
| 150,000원 | 미사일 폭격 |
| 200,000원 | 강령술 |

## 편집하기

메인 창의 "리워드 티어" 영역에서 직접 편집합니다.

1. **+ 행 추가**로 새 줄을 만들고, 금액과 기능을 지정
2. 각 줄 우측 **✕**로 해당 줄 삭제
3. **저장** 클릭 → 즉시 적용되고, 동시에 `reward_preset.json`으로 내보내기까지 완료

!!! note "금액 중복"

    같은 금액을 두 번 이상 넣으면 나중에 입력한 값으로 덮어씌워지고 로그에 경고가 남습니다.

## 프리셋 잠금

`reward_preset.json`이 한 번이라도 저장되면(또는 런처에서 프리셋을 불러오면) **편집 UI 자체가 잠깁니다** — 금액 칸, 기능 콤보박스, 행 추가/삭제, 저장 버튼이 모두 비활성화되거나 숨겨집니다. 실수로 시즌 중 티어가 바뀌는 걸 막기 위한 장치입니다.

잠금을 풀려면 런처(게이트) 화면으로 돌아가 "초기화" 버튼을 눌러야 합니다. 초기화하면 프리셋이 삭제되고 기본 티어로 되돌아갑니다.

### 프리셋 불러오기

런처 체크리스트 화면에서 **리워드 프리셋 불러오기**로 `{금액: featureId}` 형식의 JSON 파일을 미리 적용할 수도 있습니다. 형식이 맞지 않거나 존재하지 않는 featureId가 섞여 있으면 유효한 항목만 걸러서 적용합니다.

## 테스트 후원

실제 후원 없이 기능을 테스트해볼 수 있습니다.

1. 메인 창의 "테스트 후원" 드롭다운에서 저장된 티어 중 하나를 선택
2. **확인** 클릭 → 후원자명 "테스트후원자", 메시지 "테스트"로 `rewards.txt`에 즉시 기록

!!! tip

    테스트 주입도 실제 후원과 동일하게 `rewards.txt`를 거치기 때문에, 인게임에서 실제로 기능이 발동하는지까지 그대로 확인할 수 있습니다.
