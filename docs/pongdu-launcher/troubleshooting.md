# 문제 해결

## "이미 실행 중입니다"

퐁듀는 중복 실행을 막습니다. 작업 관리자에서 `PongDu.exe`가 이미 떠 있는지 확인하세요. 창이 안 보이면 작업 표시줄이나 시스템 트레이에 숨어있을 수 있습니다.

## "유효하지 않은 채널입니다"

세 가지 원인 중 하나입니다.

1. 입력한 URL/채널명/UUID로 채널을 특정할 수 없음 → 입력값 다시 확인
2. 채널은 찾았지만 이번 시즌 화이트리스트에 없음 → 시즌 참가 등록 여부 확인
3. 화이트리스트 원격 조회 자체가 실패함 (네트워크 문제) → 이 경우도 안전을 위해 차단 처리됩니다. 인터넷 연결을 확인하고 재시도하세요.

## 연동 체크리스트가 안 넘어감

**연동 시작** 버튼은 4가지가 모두 초록색이어야 활성화됩니다. 안 켜지는 항목을 확인하세요.

| 항목 | 확인할 것 |
| --- | --- |
| 생방송 중 | 방송이 실제로 켜져 있는지 |
| PZ 실행 중 | Project Zomboid가 실행되어 있는지 |
| 인게임 접속 완료 | 서버에 실제로 접속했는지 (로비/로딩 화면이 아닌 상태) |
| 19세 아님 | 방송이 연령 제한 상태가 아닌지 |

## rewards.txt 경로를 못 찾음

자동 탐지는 `~/Zomboid/Lua/rewards.txt` 및 몇 가지 후보 경로만 확인합니다. 게임을 아직 한 번도 실행하지 않았거나 세이브 폴더가 특수한 위치에 있으면 자동 탐지가 실패할 수 있습니다. 이 경우 **직접 지정** 버튼으로 폴더를 수동 선택하세요. 자세한 내용은 [rewards.txt 경로](rewards-path.md) 참고.

## 후원했는데 인게임에서 아무 반응이 없음

1. 후원 금액이 리워드 티어에 **정확히** 등록된 금액인지 확인 (이상/이하가 아니라 완전 일치)
2. 해당 featureId가 "(미구현)" 표시가 있는 기능은 아닌지 확인 — [리워드 티어](reward-tiers.md)의 전체 기능 목록 참고
3. **테스트 후원** 기능으로 같은 금액을 주입해봐서, 앱 자체 문제인지 모드 쪽 문제인지 구분

## 리워드 티어 편집이 안 됨 (잠겨 있음)

`reward_preset.json`이 이미 저장되어 있으면 편집 UI가 의도적으로 잠깁니다. 런처(게이트) 화면의 **초기화** 버튼으로 프리셋을 삭제하면 다시 편집할 수 있습니다.

## PZ 최적화가 적용되지 않음

| 증상 | 원인 / 대처 |
| --- | --- |
| "패치 리소스 없음" | 빌드 시 `--add-data "opt_conf;opt_conf"`가 빠짐 — [설치](../getting-started/installation.md) 참고 |
| "PZ 설치 폴더를 못 찾음" | Steam 비-표준 경로거나 비-Steam 버전 — 레지스트리/드라이브 스캔 범위 밖일 수 있음 |
| "게임 최적화 대기" | PZ가 실행 중이라 파일을 못 건드림 — 게임을 끄고 재시작 |
| "게임 파일 구조가 예상과 다름" | B41이 아니거나 게임 파일이 변형됨 — 안전을 위해 아무것도 패치하지 않고 중단됨 |
| 관리자 권한 창이 계속 뜸 | 게임이 `Program Files`처럼 일반 권한으로 못 쓰는 경로에 설치됨 — UAC 승격을 허용해야 적용됨 |

## 최적화를 원래대로 되돌리고 싶음

메인 화면 하단의 **해제** 버튼을 누르면 `puppet_opt_backup/`에 보관된 원본으로 복원됩니다. 이 백업 폴더 자체가 사라진 경우엔 복원이 불가능하며, Steam 라이브러리에서 해당 게임 우클릭 → **속성 → 로컬 파일 → 게임 파일 무결성 검사**로 복구해야 합니다.

## 19세 경고가 계속 뜸

방송이 실제로 연령 제한 상태인 동안은 계속 표시됩니다. 일반 방송으로 전환하면 별도 조작 없이 자동으로 사라지고 연동이 다시 가능해집니다. 수동으로 우회하는 기능(NID 쿠키 등)은 현재 버전에 없습니다.
